# Imagem do estado vazio do Histórico

Para exibir **exatamente** a ilustração do estado vazio (tela com pastas, arquivo e lupa em estilo isométrico, fundo escuro e base oval branca):

1. Copie a imagem que você tem (a ilustração fornecida) para esta pasta (`public/`).
2. **Renomeie o arquivo para:** `empty-history.png`

O arquivo deve ficar em: `modulo-eventos-novo/public/empty-history.png`

Assim, na aba **Histórico**, quando não houver nenhum registro, essa imagem aparecerá centralizada na página, com o texto "Nenhum registro no histórico." logo abaixo.
